# ArchiFlow Labs Website

## Project Documentation

### Overview
ArchiFlow Labs is a collaborative platform designed for architects and engineers to showcase their workflow and project progress.

### Features
- **User Dashboard**: Allows users to track their activities and projects.
- **Project Management**: Tools for managing design workflows including task assignments and deadlines.
- **Collaboration Tools**: Interactive tools for real-time collaboration with team members.

### Technologies Used
- Frontend: React, CSS
- Backend: Node.js, Express
- Database: MongoDB

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/oyassina101-ctrl/archiflow-labs-website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

### Usage
- Access the application through `http://localhost:3000`.
- Use the provided user credentials for initial login.
